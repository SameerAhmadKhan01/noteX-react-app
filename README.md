# NoteX 📝

NoteX is a sleek, minimalist dark-mode note-taking application built using **React**, **Vite**, and custom **CSS3**. It provides a crisp, fast user experience with seamless persistent storage so you never lose your data.

## ✨ Features

- **Persistent Storage:** Uses the Web LocalStorage API to save your notes across browser refreshes.
- **Modern UI:** Styled with a custom deep slate-gray layout featuring smooth focus indicators and animations.
- **Typography:** Enhanced readability powered by Google Fonts (**Poppins** for punchy headers and **Inter** for clean UI text).
- **Responsive Layout:** Designed to scale seamlessly for clutter-free note management.

## 🛠️ Tech Stack

- **Frontend Library:** React (Functional Components & Hooks)
- **Build Tool:** Vite (Fast Refresh & Optimized Bundling)
- **Styling:** Vanilla CSS3
- **Fonts:** Inter & Poppins (via Google Fonts)

## 🚀 Ways to Run & Deploy

### Prerequisites

Before running the application, make sure you have **Node.js** installed on your machine. You can download it from [nodejs.org](https://nodejs.org/).

### Quick Start

Follow these quick commands in your terminal to install the necessary dependencies and launch the local environment:

```bash
# Install all required packages listed in package.json
npm install

# Start the Vite development server
npm run dev
```
